# 🧮 Electron 極簡計算機 (Electron Calculator)



這是一個使用 **Electron.js**、HTML、CSS 和 JavaScript 構建的極簡桌面計算機應用程式。它旨在作為學習和演示如何將現代 Web 技術包裝成跨平台桌面應用程式的最小範例。

---

## ✨ 主要特色 (Features)

* **跨平台桌面應用程式：** 可在 Windows, macOS, 和 Linux 上運行。
* **純 Web 技術：** 應用程式介面 (UI) 完全由 HTML, CSS 和 JavaScript 構建。
* **基礎算術運算：** 支援加、減、乘、除。
* **簡潔介面：** 最小化設計，專注於核心計算功能。

---

## 🚀 快速上手 (Getting Started)

### 1. 安裝前準備

在運行此應用程式之前，你需要安裝 [Node.js](https://nodejs.org/) (包含 npm)。

* **Node.js 版本要求：** v18.0.0 或更高版本。

### 2. 下載專案

```bash
# 複製專案到你的電腦
git clone [https://github.com/你的使用者名稱/electron-calculator.git](https://github.com/你的使用者名稱/electron-calculator.git)

# 進入專案資料夾
cd electron-calculator
